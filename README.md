# Firebase CRUD Application

A simple web application demonstrating **CRUD operations** using **Firebase Firestore**.

## Features
- ✅ Add user data
- ✅ Display all users
- ✅ Edit/Update user information
- ✅ Delete users
- ✅ Real-time data management

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Firebase Firestore (NoSQL Database)

## Firebase Services Used
- Firestore Database
- Firebase SDK v12.12.1

## How to Use
1. Enter name and email
2. Click "Add User" to save
3. Use Edit/Delete buttons to modify data

## Learning Outcomes
- Firebase Firestore integration
- CRUD operations in NoSQL database
- Async/await with Firebase
- Event handling in JavaScript
